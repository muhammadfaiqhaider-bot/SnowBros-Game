# SnowBros Game 🎮

## About the Project
SnowBros is a 2D platform-style game developed in C++ using the SFML library.  
The game features progressive difficulty across multiple levels and challenging boss fights.

## Features
- 10 progressively increasing difficulty levels  
- 2 boss battles  
- Smooth player movement and collision system  
- Enemy behavior and game logic implementation  
- SFML-based graphics and rendering  

## Tech Stack
- C++
- SFML
- Object-Oriented Programming (OOP)

## Project Structure
- src/ → Source files (.cpp)
- include/ → Header files (.h)
- assets/ → Game images, sounds, and resources

## Note
This project was developed as a university group project for learning game development concepts and OOP principles.
